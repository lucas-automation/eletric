# ⚡ Motor Insulation Analysis System (IEEE 43)
### Sistema de Análise de Isolamento de Motores

This project automates the analysis of insulation resistance tests in electric motors, providing professional Excel reports and technical diagnostics based on international standards.

Este projeto automatiza a análise de testes de resistência de isolamento em motores elétricos, gerando relatórios profissionais em Excel e diagnósticos técnicos baseados em normas internacionais.

---

## 🛠 Features / Funcionalidades

* **Temperature Correction (Kt):** Automatically normalizes measurements to 40°C base according to IEEE 43.
    * *Correção de Temperatura (Kt): Normaliza automaticamente as medições para a base de 40°C conforme a norma IEEE 43.*
* **Dew Point Calculation:** Uses the Magnus-Tetens formula to alert about condensation risks.
    * *Cálculo de Ponto de Orvalho: Utiliza a fórmula de Magnus-Tetens para alertar sobre riscos de condensação.*
* **Polarization Index (PI):** Calculates and diagnoses the PI to evaluate winding contamination.
    * *Índice de Polarização (PI): Calcula e diagnostica o PI para avaliar a contaminação do enrolamento.*
* **Professional Dashboard:** Exports data to a formatted Excel file with logarithmic charts.
    * *Dashboard Profissional: Exporta os dados para um Excel formatado com gráficos logarítmicos.*

## 📊 Technical Standards / Normas Técnicas
* **IEEE 43-2013:** Recommended Practice for Testing Insulation Resistance of Electric Machinery.
